What's New in AN2592 "How to achieve 32-bit timer resolution using the link system in STM32F101xx and STM32F103xx microcontrollers" V3.0.0

The AN2592 "How to achieve 32-bit timer resolution using the link system in STM32F101xx and STM32F103xx microcontrollers" V3.0.0 is an update of V2.0.2 that makes the application note compliant with the Cortex™ microcontroller software interface standard (CMSIS).

What is AN2592 "How to achieve 32-bit timer resolution using the link system in STM32F101xx and STM32F103xx microcontrollers"?

The AN2592 "How to achieve 32-bit timer resolution using the link system in STM32F101xx and STM32F103xx microcontrollers" gives general guidelines to emulate a 32-bit timer. The two basic operating modes, that is the input capture mode and the output compare mode, are presented. Each mode is treated independently and, each time, examples of applications are provided.

The AN2592 "How to achieve 32-bit timer resolution using the link system in STM32F101xx and STM32F103xx microcontrollers" is full CMSIS compliant.
